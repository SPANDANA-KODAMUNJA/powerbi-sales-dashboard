# 📊 Power BI Sales Dashboard

## 📌 Overview
This project presents a Power BI dashboard analyzing shipment, revenue, and profit performance across products and regions.

## 📊 Key Metrics
- Shipment Count: 25K
- Total Revenue: $141M
- Total Profit: $81M
- Profit %: 57.3%

## 🚀 Features
- Monthly trend analysis (Current Year vs Previous Year)
- Top-performing products and salespersons
- Profit % indicators with conditional formatting
- Geo-based revenue distribution

## 💡 Insights
- High-performing products contribute most of the profit
- Profit % remains consistent across top salespersons
- Seasonal variation observed in shipment trends

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
